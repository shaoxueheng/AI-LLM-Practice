## 📖 内容简介

本系列从零开始讲解大模型技术，不堆砌术语，注重直觉理解与动手实践。无论你是刚入门的新手，还是希望系统补全知识体系的开发者，都能在这里找到适合自己的内容。

| 部分 | 主题 | 内容概览 |

| 基础篇 | 第1~7篇 | 大模型是什么、API调用、Python基础、深度学习、NLP入门 |

| 实战篇 | 第33~41篇 | 智能客服系统、垂直大模型、多Agent工作流、技术趋势 |
