第13章 多模态大模型应用

核心隐喻：大模型的"五感觉醒"
如果说纯文本大模型是一位博学但"闭着眼睛、捂着耳朵"的学者，那么多模态大模型就是一位真正睁开眼睛、竖起耳朵、全方位感知世界的智者。本章将带你走进大模型的"感官世界"，理解它如何同时处理文字、图像、语音和视频。

13.1 多模态大模型概述：文本、图像、语音、视频

13.1.1 什么是"多模态"

在日常生活中，人类无时无刻不在进行多模态信息处理：你看到一道菜（视觉），闻到它的香味（嗅觉），听到锅铲翻炒的声音（听觉），同时脑海中浮现出菜名和做法（语言）。这种多种感官协同工作的能力，正是多模态的本质。

对于AI而言，"模态"（Modality）指的是信息的表现形式：

表13-1　

模态 人类感官类比 数据形式 AI处理方式
文本 语言/阅读 字符串序列 Tokenizer + Transformer
图像 视觉 像素矩阵(H x W x C) ViT / CNN 编码
语音 听觉 波形/频谱图 Whisper / 声学模型
视频 动态视觉 图像帧序列 时空编码器

13.2 视觉语言模型：图片理解与生成

13.2.1 VLM 的核心思想

视觉语言模型（Vision-Language Model, VLM）的核心挑战在于：如何让"看图"和"说话"使用同一种语言？

这就像让一位画家和一位作家坐在同一张桌子前，画家把画面"翻译"成作家能理解的描述，而作家则用文字指导画家作画。VLM中的视觉编码器扮演画家，语言模型扮演作家，而连接二者的投影层就是那张"翻译桌"。

13.2.5 使用 Qwen2.5-VL 进行图像理解

# 第13章/python

from transformers import Qwen2VLForConditionalGeneration, AutoProcessor

from PIL import Image

 

# 加载模型和处理器

model = Qwen2VLForConditionalGeneration.from_pretrained(

    "Qwen/Qwen2.5-VL-7B-Instruct",

    torch_dtype="auto",

    device_map="auto"

)

processor = AutoProcessor.from_pretrained("Qwen/Qwen2.5-VL-7B-Instruct")

 

# 准备输入：图片 + 文字提问

image = Image.open("example.jpg")

messages = [

    {

        "role": "user",

        "content": [

            {"type": "image", "image": image},

            {"type": "text", "text": "请详细描述这张图片的内容，包括场景、人物、氛围。"}

        ]

    }

]

 

# 处理输入并生成

text = processor.apply_chat_template(messages, tokenize=False, add_generation_prompt=True)

inputs = processor(

    text=[text],

    images=[image],

    padding=True,

    return_tensors="pt"

).to(model.device)

 

output_ids = model.generate(**inputs, max_new_tokens=512)

response = processor.batch_decode(output_ids, skip_special_tokens=True)[0]

print(response)

# 第13章/python

import whisper

import torch

 

# 加载模型（可选：tiny, base, small, medium, large-v3）

model = whisper.load_model("large-v3")

 

# 基础转录

result = model.transcribe(

    "meeting_audio.mp3",

    language="zh",           # 指定中文

    task="transcribe",       # 转录（不翻译）

    word_timestamps=True,    # 获取词级时间戳

    initial_prompt="以下是一段关于人工智能的技术讨论。"  # 提示词引导

)

 

# 输出结果

print(f"识别文本: {result['text']}")

 

# 逐段输出（含时间戳）

for segment in result['segments']:

    start = segment['start']

    end = segment['end']

    text = segment['text']

    print(f"[{start:.1f}s - {end:.1f}s] {text}")

13.4 文生图技术：Stable Diffusion 与 DALL-E

13.4.1 扩散模型的直觉理解

扩散模型的原理可以用一个绝妙的类比来理解：

正向过程（加噪）：想象你在一杯清水中滴入一滴墨水，墨水逐渐扩散，水变得越来越浑浊，最终变成一杯均匀的"灰水"。
反向过程（去噪）：现在，模型学会了"逆转时间"，它能从那杯灰水中，一步一步地把墨水"吸"回去，恢复成清水中一滴墨水的样子 -- 只不过这次，它可以根据你的文字描述，决定墨水最终呈现什么图案。

13.4.6 实战：使用 Diffusers 生成图像

# 第13章/python

from diffusers import StableDiffusionXLPipeline

import torch

 

# 加载 SDXL 模型

pipe = StableDiffusionXLPipeline.from_pretrained(

    "stabilityai/stable-diffusion-xl-base-1.0",

    torch_dtype=torch.float16,

    variant="fp16",

    use_safetensors=True

).to("cuda")

 

# 生成图像

prompt = "一座被樱花环绕的古老中国石桥，水面倒映着粉色花瓣，水墨画风格，精致细腻"

negative_prompt = "模糊, 低质量, 变形, 文字水印"

 

image = pipe(

    prompt=prompt,

    negative_prompt=negative_prompt,

    num_inference_steps=30,     # 去噪步数

    guidance_scale=7.5,         # CFG强度

    width=1024,

    height=1024,

    generator=torch.Generator(device="cuda").manual_seed(42)

).images[0]

 

image.save("cherry_blossom_bridge.png")

print("图像已生成!")
