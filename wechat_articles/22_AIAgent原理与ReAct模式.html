12.3 ReAct 模式：推理与行动的交替

12.3.1 ReAct 是什么

ReAct（Reasoning + Acting）是 2022 年由 Yao et al. 提出的 Agent 范式。核心思想是：让 LLM 在"思考"和"行动"之间交替进行。

# 第12章/text

传统 LLM：问题 → 回答（一步到位，可能错误）

ReAct Agent：问题 → 思考 → 行动 → 观察 → 思考 → 行动 → ... → 最终回答

REACT_PROMPT = """你是一个智能助手，能使用以下工具来帮助用户完成任务：

 

可用工具：

{tools}

 

请按照以下格式回答：

 

Thought: 我需要思考接下来该怎么做

Action: 工具名称

Action Input: 工具输入参数

Observation: 工具返回的结果（由系统自动填充）

 

... （可以重复 Thought/Action/Observation 多次）

 

当你有足够信息回答时：

Thought: 我现在有足够的信息来回答了

Final Answer: 最终回答

 

开始！

 

用户问题：{question}

{agent_scratchpad}"""

12.3.4 使用 LangChain 实现 ReAct Agent

# 第12章/python

"""使用 LangChain 构建 ReAct Agent"""

from langchain_openai import ChatOpenAI

from langchain.agents import AgentExecutor, create_react_agent

from langchain_core.prompts import PromptTemplate

from langchain_core.tools import tool

 

# ===== 定义工具 =====

@tool

def search(query: str) -> str:

    """搜索互联网获取实时信息。输入为搜索关键词。"""

    # 实际项目中接入搜索 API

    return f"搜索结果：关于'{query}'的最新信息..."

 

@tool

def calculator(expression: str) -> str:

    """计算数学表达式。输入为合法的数学表达式字符串。"""

    try:

        result = eval(expression)

        return str(result)

    except Exception as e:

        return f"计算错误: {e}"

 

@tool

def get_current_time() -> str:

    """获取当前时间。无需输入参数。"""

    from datetime import datetime

    return datetime.now().strftime("%Y-%m-%d %H:%M:%S")

 

tools = [search, calculator, get_current_time]

 

# ===== 创建 Agent =====

llm = ChatOpenAI(model="gpt-4o-mini", temperature=0)

 

prompt = PromptTemplate.from_template(REACT_PROMPT)

agent = create_react_agent(llm, tools, prompt)

agent_executor = AgentExecutor(

    agent=agent,

    tools=tools,

    verbose=True,      # 打印思考过程

    max_iterations=10,  # 最大循环次数

    handle_parsing_errors=True,

)

 

# ===== 运行 Agent =====

result = agent_executor.invoke({

    "question": "今天是几号？计算距离2026年春节还有多少天。"

})

print(result["output"])

# 第12章/python

"""工具定义最佳实践"""

from langchain_core.tools import tool

from pydantic import BaseModel, Field

 

# 方式1：使用 @tool 装饰器（简单场景）

@tool

def search_web(query: str) -> str:

    """在互联网上搜索信息。

 

    Args:

        query: 搜索关键词，尽量具体明确。

 

    Returns:

        搜索结果的文本摘要。

    """

    import requests

    # 使用 Tavily 搜索 API

    response = requests.post(

        "https://api.tavily.com/search",

        json={"query": query, "max_results": 5},

        headers={"Authorization": f"Bearer {TAVILY_API_KEY}"},

    )

    results = response.json().get("results", [])

    return "\n".join(f"- {r['title']}: {r['content'][:200]}" for r in results)

 

 

# 方式2：使用 Pydantic 定义输入结构（复杂场景）

class SQLQueryInput(BaseModel):

    """SQL 查询工具的输入"""

    query: str = Field(description="要执行的 SQL 查询语句")

    database: str = Field(default="main", description="目标数据库名称")

 

@tool(args_schema=SQLQueryInput)

def query_database(query: str, database: str = "main") -> str:

    """在指定数据库中执行 SQL 查询。

    仅支持 SELECT 语句，禁止 DROP/DELETE/UPDATE 等危险操作。

    """

    # 安全检查

    dangerous = ["DROP", "DELETE", "UPDATE", "INSERT", "ALTER", "TRUNCATE"]

    if any(kw in query.upper() for kw in dangerous):

        return "错误：禁止执行修改数据的操作"

 

    import sqlite3

    conn = sqlite3.connect(f"{database}.db")

    try:

        result = conn.execute(query).fetchall()

        return str(result[:50])  # 限制返回条数

    finally:

        conn.close()
